## 📋 Overview

<a href="#tampermonkey">Tampermonkey</a> is the most widely used userscript manager, bringing the ability to run custom JavaScript on any website directly within Safari on Mac — extending what browser extensions alone can achieve by enabling community-written and self-authored scripts that modify, enhance, and automate website behavior.

The <a href="#tampermonkey">userscript installation and management</a> is Tampermonkey's core function — browsing to a userscript on Greasy Fork, OpenUserJS, or any other script repository and installing it with a single click, after which Tampermonkey runs that script automatically whenever the target website loads. Each installed script can be individually enabled, disabled, edited, or removed, giving complete control over which customizations are active on which websites. This makes the vast ecosystem of community-created userscripts — covering ad blocking, UI improvements, automation, data extraction, and thousands of other website modifications — directly accessible without requiring a dedicated extension for each individual customization. The <a href="#tampermonkey">automatic script updates</a> check userscript repositories for newer versions of installed scripts and update them automatically, ensuring scripts continue working correctly as target websites change their structure and as script authors fix bugs or add features, without requiring manual reinstallation.

The <a href="#tampermonkey">built-in script editor</a> provides a full code editor within Tampermonkey for writing custom userscripts or modifying existing ones — syntax highlighting, and the ability to test and refine scripts directly without external development tools. This makes Tampermonkey valuable not just for installing existing scripts but for users who want to write their own website customizations. The <a href="#tampermonkey">granular site permission control</a> manages exactly which websites each script can run on and what browser capabilities it can access — providing security control over what installed userscripts are permitted to do on each domain.</p>

<a href="#tampermonkey">Script scheduling and triggers</a> configure when scripts execute — on page load, on a timer, or in response to specific page events — giving userscript authors control over execution timing for scripts that need to wait for specific page conditions. <a href="#tampermonkey">Cross-device sync</a> keeps installed userscripts and their configurations synchronized across multiple Macs and other devices running Tampermonkey, maintaining a consistent customized browsing experience everywhere.

<a href="#tampermonkey">GM API support</a> provides userscripts access to Tampermonkey's Greasemonkey-compatible API for cross-origin requests, persistent storage, and other capabilities beyond what a webpage's own JavaScript context allows — enabling more powerful userscripts than would be possible without these extended capabilities. <a href="#tampermonkey">Script update notifications</a> alert when installed scripts have available updates, keeping the active userscript collection current without manual checking.</p>

## 📌 FAQ

<details>
<summary>What is a userscript?</summary>

A userscript is custom JavaScript code that runs on specific websites to modify their behavior, appearance, or functionality, written by the community or by individual users.

</details>

<details>
<summary>Where do I find userscripts to install?</summary>

Greasy Fork and OpenUserJS are the most popular userscript repositories, hosting tens of thousands of community-written scripts.

</details>

<details>
<summary>Does Tampermonkey work with Safari on Mac?</summary>

Yes. Tampermonkey is available as a Safari extension for Mac.

</details>

<details>
<summary>Are userscripts automatically updated?</summary>

Yes. Tampermonkey checks for newer versions of installed scripts from their source repositories and updates them automatically.

</details>

<details>
<summary>Can I write my own userscripts in Tampermonkey?</summary>

Yes. The built-in script editor provides a full code editor for writing and testing custom userscripts.

</details>

<details>
<summary>Is Tampermonkey free?</summary>

Tampermonkey's core userscript management functionality is free to use.

</details>

<details>
<summary>Does Tampermonkey sync across devices?</summary>

Yes. Cross-device sync keeps installed scripts and configurations consistent across multiple devices.

</details>

<details>
<summary>Does Tampermonkey send my browsing data anywhere?</summary>

No. Tampermonkey's core operation runs scripts locally without sending browsing data externally as part of its own function.

</details>

<details>
<summary>Can I control which sites a script runs on?</summary>

Yes. Granular site permission control manages exactly which websites each installed script is permitted to run on.

</details>
